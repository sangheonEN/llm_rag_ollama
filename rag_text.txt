텍스트 : 번호. 환자분 안녕하세요 : 1, 환자분 성함이 어떻게 되세요 : 2, 어디가 불편하세요 : 3, 환자분 진료 시작하겠습니다 : 4, 환자분 아 하세요 : 5, 환자분 입을 더 크게 벌려보시겠어요 : 6, 통증이 느껴진다면 왼쪽 팔을 들어주세요 : 7, 환자분 통증이 느껴지시나요 : 8, 환자분 다시 진료 시작해도 될까요 : 9, 환자분 진료 끝났습니다. 수고하셨습니다 : 10. 이와 같은 형식으로 특정 텍스트에 맞춘 번호를 정의하는 문장들이 있어. 내가 "환자분 안녕하세요"라고 입력해주면, 해당 텍스트에 매칭되는 번호를 알려줘. 문구가 조금 부정확하더라도 의미가 유사하면 번호를 알려줘.
